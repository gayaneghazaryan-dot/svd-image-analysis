# SVD for Image Compression and Denoising

This repository accompanies the paper *"Singular Value Decomposition in Data Science: Applications in Image Compression, Denoising, and Dimensionality Reduction"*.

It provides **reusable, modular code** for applying SVD to images, with examples on both **public datasets** and **custom images** (e.g., `Aren.jpg`).

## 📁 Structure
- `manuscript/` — Paper and appendix code (public data only)
- `src/` — Core SVD functions
- `supplementary/` — Code for personal images
- `notebooks/` — Interactive Jupyter demo
- `data_preview/` — Optional preview image

## 🚀 Quick Start
```bash
pip install -r requirements.txt
python supplementary/svd_on_aren.py
